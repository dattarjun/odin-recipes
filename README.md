# Odin Recipes 🍽️

A basic recipe website created as the first project of The Odin Project Foundations Course.

## 📚 About The Project

This project focuses on practicing:

- Basic HTML structure
- Headings and paragraphs
- Lists
- Links and navigation
- Images
- File structure management
- Git and GitHub workflow
- GitHub Pages deployment

## 🍛 Recipes Included

- Maharashtrian Misal Pav
- Bhel
- Paneer Tikka Masala
- Malai Matar Methi
- Pav Bhaji

## 🛠️ Technologies Used

- HTML5
- Git
- GitHub
- GitHub Pages
